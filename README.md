ParkiMove is an interactive Parkinson’s rehabilitation web game designed to support hand movement, coordination, and motor-control practice through simple therapy-based exercises. The website includes exercises such as Grip & Place, Path Tracing, Finger Isolation, and Tremor Control, with progressive levels, real-time feedback, scoring, and session progress tracking. The goal of this project is to make rehabilitation practice more engaging by combining game-like interaction with hand-tracking features, helping users practice grip control, finger movement, path accuracy, and hand stability in a clear and accessible way.
